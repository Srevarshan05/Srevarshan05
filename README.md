### `AI Nose` — Edge Olfactory Classification System

> **Stack:** ESP32 · Gas Sensor Array · Random Forest · Flask

An IoT device that classifies Volatile Organic Compounds in real-time using a gas sensor array. Random Forest classifier deployed on-device — no cloud dependency, 92% accuracy. Built to understand how inference performs under hardware constraints. *(Lightweight systems work — and knowing why matters for AI deployment.)*

---

### `E-Uzhavan` — Predictive Crop Monitoring

> **Stack:** IoT Sensor Network · Cloud Analytics · Predictive Modeling

Distributed sensor network streaming soil metrics to a cloud dashboard. Predictive irrigation algorithms recommend watering schedules — modeled to cut water use by ~20%. Designed for scale, built for farmers who operate on thin margins.

### `Manufacturing Defect Detection` — Microsoft × Edunet Internship

> **Stack:** YOLO v8n · Azure Cloud · Raspberry Pi · Computer Vision

Custom dataset. Custom-trained YOLO v8n model. Serverless Azure endpoint. Live inference on a production line via Raspberry Pi camera. From data collection to deployed endpoint — built and shipped within an internship.
