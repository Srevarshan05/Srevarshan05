## Projects

### [Aura-Sync](https://github.com/Srevarshan05/Aura-Sync) — Collaborative Cloud Arduino IDE
`Node.js` `Docker` `WebSockets` `Arduino CLI` `GCP`

Real-time collaborative IDE for Arduino hardware — built before Arduino officially launched its own cloud editor. Multiple engineers, one shared session, sandboxed cloud compilation, sub-100ms sync. Think Google Docs, but it flashes firmware. Deployed on GCP.

---

### [AgrosCare](https://github.com/Srevarshan05) — Cotton Crop Real-Time Monitoring
`IoT Sensor Probe` `LoRa` `FOG Computing` `Firebase RTDB`

A high-precision multi-sensor probe system for real-time cotton crop monitoring and infestation alerts. Uses FOG computing to bridge internet-less fields to the cloud via LoRa. **Top 8 nationally at NAF Uzhavu Hackathon conducted by IITM. Selected for IIM product mentorship.**

---

### [VocSense](https://github.com/Srevarshan05) — Edge Olfactory Classifier
`ESP32` `Gas Sensor Array` `Random Forest` `Flask`

IoT device that classifies Volatile Organic Compounds in real-time using a gas sensor array. Random Forest classifier deployed on-device — no cloud, no latency, 92% accuracy.

---

### [Manufacturing Defect Detection](https://github.com/Srevarshan05) — Microsoft × Edunet
`YOLO v8n` `Azure Cloud` `Raspberry Pi` `Computer Vision`

Custom-trained YOLO v8n model on a curated bottle cap defect dataset. Deployed as a serverless Azure endpoint, integrated with Raspberry Pi camera for live inference on a production line.
